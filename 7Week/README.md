# 7 WEEK

7주차 과제입니다.

## Flexbox gallery

1. '학번_이름' 폴더를 '7Week' 폴더에 생성합니다.
2. '학번_이름' 폴더에 Flexbox 스타일을 이용한 이미지 갤러리를 제작하여 업로드합니다.
